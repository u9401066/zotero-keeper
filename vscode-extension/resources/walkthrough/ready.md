## Try These Commands

Open **GitHub Copilot Chat** and try:

### 🔍 Search Literature
> "Search PubMed for machine learning in radiology"

### 📚 Browse Zotero
> "Show my recent references in Zotero"

### 💾 Save References
> "Save PMID 12345678 to my Zotero library"

### 📊 Get Citation Metrics
> "What's the citation count for this paper?"

## Available MCP Tools

| Tool | Description |
|------|-------------|
| `unified_search` | Search PubMed and other academic sources |
| `generate_search_queries` | Build MeSH queries |
| `search_items` | Search Zotero |
| `import_articles` | Import articles into Zotero |
| `check_articles_owned` | Check if articles already in Zotero |
